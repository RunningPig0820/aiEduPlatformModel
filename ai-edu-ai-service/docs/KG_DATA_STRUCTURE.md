## 3. 节点类型

### 3.1 Entity (知识点实体)

**用途**: 表示一个具体的知识点

**属性**:

| 属性名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `uri` | string | 唯一标识符 | `http://edukg.org/knowledge/0.1/instance/math#xxx` |
| `label` | string | 知识点名称 | `一元二次方程` |
| `subject` | string | 所属学科 | `math` |

**示例数据**:

```json
{
  "uri": "http://edukg.org/knowledge/0.1/instance/math#equation",
  "label": "一元二次方程",
  "subject": "math"
}
```

### 3.2 Class (分类节点)

**用途**: 表示知识点的分类/类别

**属性**:

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `uri` | string | 唯一标识符 |
| `label` | string | 分类名称 |

### 3.3 Student (学生节点) - 动态创建

**用途**: 存储学生学习进度

**属性**:

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `id` | string | 学生ID |

## 5. 前端渲染建议

### 5.1 知识树结构

**数据格式** (调用 `/api/kg/subject/{subject}/tree`)：

```
数学 (root)
├── 代数 (class)
│   ├── 一元二次方程 (entity)
│   ├── 二次函数 (entity)
│   └── 一元一次方程 (entity)
├── 几何 (class)
│   ├── 三角形 (entity)
│   ├── 圆 (entity)
│   └── 四边形 (entity)
└── 统计与概率 (class)
    ├── 平均数 (entity)
    └── 概率 (entity)
```

**渲染组件推荐**:

| 组件库 | 组件 | 适用场景 |
|--------|------|----------|
| Ant Design | Tree | 简单树形列表 |
| ECharts | Tree | 可视化树图 |
| D3.js | Tree/Cluster | 复杂交互树 |
| AntV G6 | Graph | 关系图谱 |

### 5.2 知识图谱网络

**数据格式** (Entity + Relationships)：

```
┌─────────────┐
│ 一元二次方程 │
└──────┬──────┘
       │ subClassOf
       ▼
┌─────────────┐    leftRelate    ┌─────────────┐
│   方程      │ ───────────────▶ │   函数      │
└─────────────┘                  └─────────────┘
```

**渲染建议**:

```
节点样式:
┌─────────────────────────────────────┐
│  [学科标签]  一元二次方程            │
│  学习状态: ████████░░ 已掌握 80%    │
└─────────────────────────────────────┘

颜色方案:
- math: 蓝色 (#1890ff)
- physics: 紫色 (#722ed1)
- chemistry: 绿色 (#52c41a)
- biology: 青色 (#13c2c2)
- chinese: 红色 (#f5222d)
- history: 橙色 (#fa8c16)
- geo: 青绿 (#2f9e8d)
- politics: 棕色 (#874d00)
- english: 粉色 (#eb2f96)
```

### 5.3 学习进度展示

**状态颜色**:

## 7. API 调用场景

### 7.1 知识图谱首页

```
调用接口:
1. GET /api/kg/subject/{subject}/tree    → 获取知识树
2. GET /api/kg/subject/{subject}/classes → 获取分类

渲染:
- 左侧: 学科选择器 (9个学科)
- 中间: 知识树 (可展开/折叠)
- 右侧: 知识点详情
```

### 7.2 知识点搜索

```
调用接口:
1. GET /api/kg/entities?label={keyword}  → 搜索知识点

渲染:
- 搜索框下拉联想
- 显示学科标签
- 点击跳转详情页
```

### 7.3 知识点详情页

```
调用接口:
1. GET /api/kg/entity/{uri}              → 获取详情
2. GET /api/kg/recommend?entity_uri={uri} → 获取推荐

渲染:
- 知识点名称
- 相关知识点 (关系图)
- 前置知识 (学习路径)
- 相关推荐
```

### 7.4 学习进度展示

```
调用接口:
1. GET /api/kg/student/{id}/progress     → 获取进度
2. GET /api/kg/student/{id}/statistics   → 获取统计

渲染:
- 进度条/仪表盘
- 各学科掌握情况
- 学习建议
```

## 8. 数据请求示例

### 8.1 获取数学知识树

```bash
curl -X GET "http://localhost:9527/api/kg/subject/math/tree?depth=3" \
  -H "x-internal-token: your-token"
```

**响应**:

```json
{
  "subject": "math",
  "tree": {
    "id": "root",
    "label": "数学",
    "type": "class",
    "subject": "math",
    "children": [
      {
        "id": "http://edukg.org/knowledge/xxx",
        "label": "一元二次方程",
        "type": "entity",
        "subject": "math",
        "children": []
      }
    ]
  }
}
```

### 8.2 搜索知识点

```bash
curl -X GET "http://localhost:9527/api/kg/entities?label=方程&subject=math&limit=10" \
  -H "x-internal-token: your-token"
```

**响应**:

```json
{
  "total": 5,
  "entities": [
    {"label": "一元二次方程", "uri": "...", "subject": "math"},
    {"label": "二元一次方程", "uri": "...", "subject": "math"},
    {"label": "分式方程", "uri": "...", "subject": "math"}
  ]
}
```

## 9. 注意事项

### 9.1 数据特点

1. **知识点名称不唯一**: 同一知识点可能在不同学科出现
2. **URI 是唯一标识**: 使用 URI 作为知识点唯一标识
3. **学科代码统一**: 使用英文代码 (math, physics 等)

### 9.2 性能建议

1. **缓存知识树**: 知识树结构不常变化，建议缓存
2. **分页加载**: 大量数据时使用 limit 参数分页
3. **按学科加载**: 不要一次性加载所有学科数据

### 9.3 前端交互建议

```
1. 学科选择器
   - 默认选中第一个学科
   - 切换学科时重新加载知识树

2. 知识树
   - 默认展开前2层
   - 懒加载子节点 (如果数据量大)
   - 点击节点显示详情

3. 搜索
   - 输入防抖 (300ms)
   - 最少2个字符才开始搜索

4. 学习进度
   - 颜色区分状态
   - 进度条/饼图展示
```
