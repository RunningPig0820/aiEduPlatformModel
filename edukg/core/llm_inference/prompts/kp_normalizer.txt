# 知识点标准化提示词

将教材知识点名称推断为标准数学概念名称，提高与 EduKG 的匹配率。

## 输入变量

- `{kp_name}`: 教材知识点名称
- `{stage}`: 学段（小学/初中/高中）
- `{grade}`: 年级

## 提示词模板

```
你是一位数学教育专家。你的任务是将教材知识点名称转换为**标准数学概念名称**，以便与知识图谱（EduKG）中的概念进行匹配。

## 输入
教材知识点名称：{kp_name}
学段：{stage}
年级：{grade}

## EduKG 常见概念术语（优先使用）
以下术语在 EduKG 知识图谱中真实存在，优先使用可提高匹配成功率：

### 数与运算（高频）
- 基础：数、自然数、整数、实数、分数、小数、百分数
- 运算：加法、减法、乘法、除法、混合运算、四则运算、乘方
- 关系：大于、小于、等于、比例、比

### 图形与几何（高频）
- 基础：点、线、面、角、三角形、四边形、圆、平行、垂直
- 性质：对称、全等、相似、周长、面积、体积
- 特殊：直角、平行线、垂线、中点、切点

### 方程与函数（高频）
- 方程：方程、一元一次方程、一元二次方程、二元一次方程组
- 函数：函数、一次函数、二次函数、三角函数、指数函数

### 统计与概率（高频）
- 统计：统计、统计图、平均数、中位数、抽样
- 概率：概率、随机事件、分布

## 粒度控制（根据学段年级）
- 小学1-2年级：使用基础概念（自然数、加法、角、三角形）
- 小学3-6年级：可使用复合概念（分数运算、面积计算、混合运算）
- 初中：使用标准代数/几何术语（一元一次方程、平行四边形）
- 高中：使用抽象数学术语（函数、向量、导数）

## 推断规则
1. "认识"、"初步"类 → 推断核心概念（"1-5的认识" → "自然数"）
2. "组成"、"分解"类 → 推断关系概念（"6和7的组成" → "数"或"加法"）
3. "性质"、"特征"类 → 推断主体概念（"平行四边形的性质" → "平行四边形")
4. "解决问题"类 → 推断涉及的运算或模型

## 输出要求
输出 JSON 对象，包含以下字段：
- `concepts`: 推断的标准概念名称列表，按可能性从高到低排序，最多5个
- `best_match`: 最可能匹配的单一概念名称（用于向量检索）
- `confidence`: 推断置信度（0.0-1.0），基于名称清晰度和概念匹配度
- `reason`: 简要推断理由（不超过30字）

## 示例
输入：
教材知识点名称：1-5的认识
学段：小学
年级：一年级

输出：
{{"concepts": ["自然数", "数", "整数"], "best_match": "自然数", "confidence": 0.9, "reason": "认识1-5对应自然数基础概念"}}

教材知识点名称：连加连减
学段：小学
年级：一年级

输出：
{{"concepts": ["加法", "减法", "混合运算"], "best_match": "加法", "confidence": 0.85, "reason": "连加连减包含加法和减法运算"}}

输入：
教材知识点名称：秒的认识
学段：小学
年级：三年级

输出：
{{"concepts": ["时间", "秒"], "best_match": "秒", "confidence": 0.95, "reason": "秒是时间单位概念"}}

输入：
教材知识点名称：平行四边形的性质
学段：初中
年级：八年级

输出：
{{"concepts": ["平行四边形", "四边形", "平行"], "best_match": "平行四边形", "confidence": 0.9, "reason": "直接对应平行四边形概念"}}

输入：
教材知识点名称：一元二次方程的解法
学段：初中
年级：九年级

输出：
{{"concepts": ["一元二次方程", "方程"], "best_match": "一元二次方程", "confidence": 0.95, "reason": "直接对应一元二次方程概念"}}

## 重要：只输出 JSON，不要添加其他解释。
```
